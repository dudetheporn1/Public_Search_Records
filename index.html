Building the APK

  1. Clone the repository

    git clone https://github.com/dudetheporn1/Public_Search_Records.git

  2. Navigate to the project directory

    cd Public_Search_Records

  3. Ensure you have the Android SDK and Java Development Kit installed

    Android Studio is recommended for the easiest setup

  4. Build the APK using Gradle

    ./gradlew assembleDebug

    For a release build, use:

    ./gradlew assembleRelease

  5. Find the APK file

    The generated APK will be located at:

    app/build/outputs/apk/debug/app-debug.apk

    Or for release build:

    app/build/outputs/apk/release/app-release.apk

Common Issues

Missing SDK Components

If you encounter errors about missing SDK components, use Android Studio's SDK Manager to install them.

Gradle Version Mismatch

If gradle versions don't match, use the provided gradle wrapper by using ./gradlew instead of direct gradle commands.

Signing Configuration for Release

For release builds, you'll need to configure a signing key in the build.gradle file.
